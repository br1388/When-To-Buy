Esta aplicacion de consola permite pegar el link de cualquier producto de Mercado Libre (o cargar los datos manualmente) y el programa se encarga de archivarlo en una base de datos con su precio en pesos, en dólar informal y su fecha.
De esta forma permite un seguimiento a través del tiempo para detectar verdaderas ofertas, es decir, productos cuyo precio convertido a dólar informal esté más bajo que su valor habitual.
La cotización del dólar está siempre actualizada a través de una API de un diario conocido.
