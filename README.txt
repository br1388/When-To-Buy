Esta aplicacion de consola permite pegar el link de cualquier producto de Mercado Libre (o cargar los datos manualmente) y el programa se encargará de almacenar en una base de datos (SQL) su precio en pesos, en dólar informal y su fecha de carga.
Una vez registrado el o los productos en la base de datos, permite al usuario realizar un análisis temporal de la variación de su precio con relación a la cotización del dolar informal actualizado, además de permitirle consultar otras variables de interés.
Haciendo este seguimiento se pueden detectar los momentos adecuados para la compra de determinado bien, de allí su denominación, "When to buy".
Los valores de cotización se obtienen a través de una API del diario Ámbito Financiero.
